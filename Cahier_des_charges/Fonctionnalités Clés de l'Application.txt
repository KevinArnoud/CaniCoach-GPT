Fonctionnalités Clés de l'Application
1. Authentification Utilisateur
Inscription : Création de compte via email et mot de passe sécurisé.
Connexion : Accès au compte via email et mot de passe.
Déconnexion : Possibilité de se déconnecter de manière sécurisée.
Mot de passe oublié : Procédure de réinitialisation du mot de passe par e-mail.
2. Gestion des Profils Chiens
Création : L'utilisateur peut créer le profil d'au moins un chien.
Champs du Profil : Nom, photo (upload), race ou croisement, date de naissance, provenance (élevage, refuge, particulier...), champ texte pour les problèmes de santé connus, champ texte pour le passé (traumatismes, etc.).
Modification : L'utilisateur doit pouvoir modifier les informations de son chien à tout moment.
Multi-profils (pour les abonnés) : Les utilisateurs avec un abonnement actif peuvent ajouter et gérer plusieurs profils de chiens.
3. Chatbot Conversationnel
Interface de Chat : Interface principale de l'application où l'utilisateur dialogue avec CaniCoach IA.
Historique : L'utilisateur doit pouvoir accéder à l'historique de ses conversations passées.
4. Gestion de l'Abonnement
Page de Statut : Un écran où l'utilisateur peut voir son statut d'abonnement actuel (Essai, Mensuel, Annuel).
Paiement : Intégration d'une solution de paiement sécurisée comme Stripe.
Gestion : L'utilisateur doit pouvoir mettre à jour ses informations de paiement et résilier son abonnement facilement.
